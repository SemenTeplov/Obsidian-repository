| Определение | Исходный веб-фреймворк Spring Web MVC, входящий в состав Spring Framework, был специально разработан для API сервлетов и сервлет-контейнеров. Веб-фреймворк Spring WebFlux, основанный на реактивном стеке, был добавлен позже, в версии 5.0. Он полностью неблокирующий, поддерживает реактивные потоки с обратным давлением и работает на таких серверах, как Netty, а также на сервлет-контейнерах. |
