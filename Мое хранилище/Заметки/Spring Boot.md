| ----------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Определение | Spring Boot позволяет легко создавать автономные приложения на базе Spring промышленного уровня, которые можно «просто запустить». Мы придерживаемся собственного подхода к платформе Spring и сторонним библиотекам, чтобы вы могли приступить к работе с минимальными хлопотами. Для большинства приложений Spring Boot требуется минимальная настройка Spring.                                                                                                                                                               |
| Особенности | - Создавайте автономные Spring-приложения  <br>- Встраивайте Tomcat, Jetty или Undertow напрямую (без необходимости развертывать WAR-файлы)  <br>- Предоставляйте готовые «стартовые» зависимости, чтобы упростить конфигурацию сборки  <br>- По возможности автоматически настраивайте Spring и сторонние библиотеки  <br>- Предоставляйте готовые к использованию в продакшене функции, такие как метрики, проверки работоспособности и внешняя конфигурация  <br>- Никакой генерации кода и необходимости в XML-конфигурации |
