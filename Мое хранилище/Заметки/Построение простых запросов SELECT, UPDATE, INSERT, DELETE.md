| Под темы    | Заметка                                                                                                                  |
| ----------- | ------------------------------------------------------------------------------------------------------------------------ |
| Определение | SELECT, UPDATE, INSERT, DELETE - запросы SQL, которые используются для извлечения, изменения, вставки и удаления данных. |
| SELECT      | Извлечение данных из базы данных.                                                                                        |
|             | Пример: `SELECT col1, col2, ... FROM table`.                                                                             |
| UPDATE      | Обновление данных в базе данных.                                                                                         |
|             | Пример: `UPDATE table SET col1 = val1, col2 = val2, ... WHERE condition`.                                                |
| INSERT      | Вставка данных в базу данных.                                                                                            |
|             | Пример: `INSERT INTO table (col1, col2, ...) VALUES (val1, val2, ...)`.                                                  |
| DELETE      | Удаление данных из базы данных.                                                                                          |
|             | Пример: `DELETE FROM table WHERE condition`.                                                                             |

### Какой SQL запрос SELECT, который выберет все записи из таблицы employees.

| Под темы | Заметка                                                                    |
| -------- | -------------------------------------------------------------------------- |
| Запрос   | `SELECT * FROM employees`                                                  |
|          | `SELECT *` — указывает, что нужно вернуть все столбцы.                     |
|          | `FROM employees` — указывает таблицу, из которой будут извлекаться данные. |

### Какой SQL запрос UPDATE изменит зарплату сотрудника с ID 100 на 5000?

| Под темы | Заметка                                                                                                                                                                      |
| -------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Запрос   | `UPDATE employees SET salary = 5000 WHERE id = 100`                                                                                                                          |
|          | `UPDATE имя_таблицы` — таблица, в которой нужно изменить данные.                                                                                                             |
|          | `SET` — ключевое слово для указания столбцов и новых значений.                                                                                                               |
|          | `имя_столбца1 = новое_значение1` — указывает столбец и новое значение, которое в него будет записано. Можно обновлять сразу несколько столбцов, перечисляя их через запятую. |
|          | `WHERE` — условие, определяющее, какие строки нужно изменить. Если WHERE не используется, обновятся все строки таблицы.                                                      |

### Какой SQL запрос INSERT, который добавит нового сотрудника с именем "Anton" в таблицу employees.

| Под темы | Заметка                                                                          |
| -------- | -------------------------------------------------------------------------------- |
| Запрос   | `INSERT INTO employees (name) VALUES ('Anton')`                                  |
|          | `INSERT INTO table_name` — таблица, в которую выполняется вставка данных.        |
|          | `column1, column2, ...` — список столбцов, в которые будут вставлены значения.   |
|          | `value1, value2, ...` — список значений, которые соответствуют столбцам таблицы. |

### Какой SQL запрос DELETE удалит запись сотрудника с ID 200 из таблицы employees?

| Под темы | Заметка                                                     |
| -------- | ----------------------------------------------------------- |
| Запрос   | `DELETE FROM employees WHERE id = 200`                      |
|          | `DELETE FROM employees` - удаление из таблицы.              |
|          | `WHERE id = 200` - условие, которое должно быть выполненно. |

### Какой SQL запрос SELECT, который выберет имена и зарплаты всех сотрудников, у которых зарплата выше средней зарплаты в компании.

| Под темы | Заметка                                                                                           |
| -------- | ------------------------------------------------------------------------------------------------- |
| Запрос   | `SELECT name, position, salary FROM employees WHERE salary > (SELECT AVG(salary) FROM employees)` |
|          | `SELECT name, position, salary` - извлечение столбцов.                                            |
|          | `FROM employees` - извлечение из таблицы.                                                         |
|          | `WHERE salary > (SELECT AVG(salary) FROM employees)` - условие, что зарплата больше среднего.     |

### Какой SQL запрос SELECT, который выберет имена и зарплаты всех сотрудников, у которых зарплата выше средней зарплаты в их отделе.
