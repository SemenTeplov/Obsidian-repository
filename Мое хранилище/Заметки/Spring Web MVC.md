| Определение | Spring Web MVC — это оригинальный веб-фреймворк, созданный на основе API сервлетов и с самого начала включенный в Spring Framework. Официальное название — Spring Web MVC — происходит от названия исходного модуля (spring-webmvc), но чаще его называют Spring MVC. |
