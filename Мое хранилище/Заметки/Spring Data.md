| Определение           | Задача Spring Data — предоставить привычную и единообразную модель программирования на основе Spring для доступа к данным, сохранив при этом особенности базового хранилища данных.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
|                       | Это упрощает использование технологий доступа к данным, реляционных и нереляционных баз данных, фреймворков MapReduce и облачных сервисов для работы с данными. Это комплексный проект, включающий в себя множество подпроектов, ориентированных на конкретную базу данных. Проекты разрабатываются совместно со многими компаниями и разработчиками, стоящими за этими передовыми технологиями.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| Особенности           | - Мощный репозиторий и пользовательские абстракции для сопоставления объектов  <br>- Динамическое формирование запросов на основе названий методов репозитория  <br>- Базовые классы предметной области, предоставляющие основные свойства  <br>- Поддержка прозрачного аудита (создание, последнее изменение)  <br>- Возможность интеграции пользовательского кода репозитория  <br>- Простая интеграция с Spring через JavaConfig и пользовательские пространства имен XML  <br>- Расширенная интеграция с контроллерами Spring MVC  <br>- Экспериментальная поддержка кросс-хранилищной персистентности                                                                                                                                                                                                                                                                                                                                                                                                                                |
| Основные модули       | Spring Data Commons — основные концепции Spring, лежащие в основе каждого модуля Spring Data. Spring Data JDBC — поддержка репозиториев Spring Data для JDBC. Spring Data R2DBC — поддержка репозиториев Spring Data для R2DBC. Spring Data JPA — поддержка репозиториев Spring Data для JPA. Spring Data KeyValue — репозитории на основе Map и SPI для простого создания модуля Spring Data для хранилищ «ключ-значение». Spring Data LDAP — поддержка репозитория Spring Data для Spring LDAP. Spring Data MongoDB — поддержка объектно-документированных баз данных на основе Spring и репозиториев для MongoDB. Spring Data Redis — простая настройка и доступ к Redis из приложений Spring. Spring Data REST — экспорт репозиториев Spring Data в виде гипермедийных RESTful-ресурсов. Spring Data для Apache Cassandra — простая настройка и доступ к Apache Cassandra или крупномасштабным, высокодоступным, ориентированным на данные приложениям Spring.                                                                        |
| Модули сообщества     | Spring Data Aerospike — модуль Spring Data для Aerospike. Spring Data ArangoDB — модуль Spring Data для ArangoDB. Spring Data Couchbase — модуль Spring Data для Couchbase. Spring Data Azure Cosmos DB — модуль Spring Data для Microsoft Azure Cosmos DB. Spring Data Cloud Datastore — модуль Spring Data для Google Datastore. Spring Data Cloud Spanner — модуль Spring Data для Google Spanner. Spring Data DynamoDB — модуль Spring Data для DynamoDB. Spring Data Elasticsearch — модуль Spring Data для Elasticsearch. Spring Data Hazelcast — поддержка репозиториев Spring Data для Hazelcast. Spring Data Neo4j — поддержка объектно-ориентированных графов на основе Spring и репозиториев для Neo4j. Oracle NoSQL Database SDK для Spring Data — модуль Spring Data для Oracle NoSQL Database и Oracle NoSQL Cloud Service. Spring Data Reindexer — модуль Spring Data для базы данных Reindexer. Spring Data Vault — репозитории Vault на основе Spring Data KeyValue. Spring Data Valkey — модуль Spring Data для Valkey. |
| Архивированные модули | Spring Data для Apache Geode — простая настройка и доступ к Apache Geode для высокостабильных, ориентированных на данные Spring-приложений с низкой задержкой. Spring Data для GemFire — простая настройка и доступ к GemFire для высокостабильных, ориентированных на данные Spring-приложений с низкой задержкой.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
