| Под темы    | Заметка                                                                                                                                                                                                                                                                                        |
| ----------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Определение | Kubernetes, также известный как K8s, — это система с открытым исходным кодом для автоматизации развёртывания, масштабирования и управления контейнерными приложениями. Она группирует контейнеры, из которых состоит приложение, в логические единицы для удобства управления и обнаружения.   |
| Особенности | Kubernetes постепенно внедряет изменения в ваше приложение или его конфигурацию, отслеживая работоспособность приложения, чтобы не отключать все экземпляры одновременно. Если что-то пойдет не так, Kubernetes откатит изменения.                                                             |
|             | Не нужно вносить изменения в приложение, чтобы использовать незнакомый механизм обнаружения сервисов. Kubernetes предоставляет подам собственные IP-адреса и единое DNS-имя для группы подов, а также может распределять нагрузку между ними.                                                  |
|             | Автоматическое подключение выбранной вами системы хранения данных, будь то локальное хранилище, общедоступное облачное хранилище или сетевая система хранения данных, например iSCSI или NFS.                                                                                                  |
|             | Развертывайте и обновляйте секреты и конфигурацию приложений без перестроения образа и без раскрытия секретов в конфигурации стека.                                                                                                                                                            |
|             | Автоматически размещает контейнеры в соответствии с их требованиями к ресурсам и другими ограничениями, не снижая при этом доступность.                                                                                                                                                        |
|             | Помимо сервисов, Kubernetes может управлять пакетными задачами и задачами непрерывной интеграции, при необходимости заменяя вышедшие из строя контейнеры.                                                                                                                                      |
|             | Kubernetes перезапускает контейнеры, которые аварийно завершают работу, при необходимости заменяет целые поды, переподключает хранилище в случае более масштабных сбоев и может интегрироваться с системами автоматического масштабирования узлов для самовосстановления даже на уровне узлов. |
|             | Автоматическая настройка запросов и ограничений ресурсов в соответствии с фактическими моделями использования.                                                                                                                                                                                 |
|             | Распределение адресов IPv4 и IPv6 между подами и сервисами.                                                                                                                                                                                                                                    |
