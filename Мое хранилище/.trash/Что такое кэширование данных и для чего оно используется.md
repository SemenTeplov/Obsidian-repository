| https://www.mvideo.ru/blog/pomogaem-razobratsya/chto-takoe-kesh-i-dlya-chego-on-nuzhen | **Кэширование данных** — это процесс сохранения копии данных в специальном, быстродоступном хранилище (кэше), чтобы последующие запросы к этим данным можно было обработать быстрее. **Цель кэширования** — сократить время ответа и снизить нагрузку на основной источник данных.                                                                                                                                                                                                                                 |
| https://cms.by/blog/1c/kak-uskorit-rabotu-1c-s-pomoshchyu-keshirovaniya/               | **Кэширование используется в различных областях**, например:<br><br>- **Веб-сайты и веб-приложения** — ускорение загрузки страниц, снижение нагрузки на сервер.<br>- **Мобильные приложения** — кэширование данных на устройстве для пользователей, чтобы обеспечить работу в офлайн-режиме и ускорить загрузку контента.<br>- **Социальные сети** — кэширование лент новостей, профилей пользователей, списков друзей.<br>- **E-commerce** — кэширование карточек товаров, каталогов, цен и информации о наличии. |
| https://blog.click.ru/glossary/keshirovanie/                                           | **Принцип работы**<br><br>Когда система запрашивает данные, **сначала проверяется наличие их в кэше**. Если данные есть, они возвращаются из кэша, что быстрее, чем обращение к источнику данных. Если данных нет в кэше, они загружаются из источника и сохраняются в кэше для последующего использования.                                                                                                                                                                                                        |
|                                                                                        | **Виды**<br><br>Выделяют два основных способа кэширования:<br><br>1. **Клиентское (браузерное)** — данные кэшируются в браузере на стороне пользователя (изображения, стили, скрипты).<br>2. **Серверное** — данные кэшируются на стороне сервера (например, результаты запросов к базе данных).                                                                                                                                                                                                                   |
