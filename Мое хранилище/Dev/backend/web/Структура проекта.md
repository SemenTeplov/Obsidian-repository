| Под темы   | Заметка                                                                                    |
| ---------- | ------------------------------------------------------------------------------------------ |
| Entity     | Сущность, взаимодействует с базой данных. Переносит данные из базы данных в код и обратно. |
| DTO        | Сущность, служит для транспортировки данных между слоями программы.                        |
| Mapper     | Вспомогательный объект, описывающий переходы данных из одной сущности в другую.            |
| Service    | Описывает бизнес логику.                                                                   |
| Repository | Описывает взаимодействие с базами данных.                                                  |
| Controller | Описывает взаимодействие приложения с внешними источниками данных.                         |
